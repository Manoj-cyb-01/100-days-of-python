# 100 Days of Python 🐍

This repository contains my projects, exercises, and coding challenges completed during the **100 Days of Code – Python Bootcamp** by Dr. Angela Yu.

The goal of this challenge is to strengthen Python programming skills through consistent daily practice and by building practical projects.

---

## 📚 About the Course

The course focuses on learning Python by building real-world projects and applications. It covers both fundamental and advanced topics in Python programming.

Key topics include:

* Python fundamentals
* Data types and control flow
* Functions and loops
* Object-Oriented Programming
* File handling
* Working with APIs
* Automation using Python
* GUI development
* Game development
* Data handling and visualization
