# Day 02 – Tip Calculator 💰

## Overview

This project implements a simple **Tip Calculator** in Python.
The program helps users calculate how much each person should pay when splitting a bill, including a chosen tip percentage.

---

## Problem Statement

When a group of people share a meal, calculating the correct amount each person needs to pay can be tedious. This program automates the process by taking the total bill, tip percentage, and number of people, then computing the split amount.

---

## How It Works

1. The user enters the **total bill amount**.
2. The user selects the **tip percentage** (e.g., 10%, 12%, or 15%).
3. The user enters the **number of people sharing the bill**.
4. The program calculates the **final amount each person should pay**.
